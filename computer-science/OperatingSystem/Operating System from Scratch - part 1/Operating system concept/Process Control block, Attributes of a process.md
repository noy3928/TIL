### 동적 메모리 할당이란

런타임 중에 메모리를 할당하는 것을 동적 메모리 할당이라고 한다.

일반적으로 힙과 스택은 동등하게 분배되지 않는다는 것을 이해하고 있자.
이것이 분배되는 것은 오직 프로그램에 따라서 달라지게 되는 것이다.

<br>

## 프로세스의 특성

1. process id :  
   오직 하나의 프로세스에만 부여되는 아이디 값. unique한 값이다.

2. program counter :
   이 프로그램 카운터에 의해서, 프로세스가 스케줄링 되면서 순서가 뒤바뀔 때 어디서 시작해야하는지 알 수 있게 된다. 이것은 레지스터의 일종이라고 생각하면 된다.

- cpu 스케줄링이란, 단지 어떤 프로세스를 먼저 실행것인가. 프로세스를 어떤 순서로 실행할 것인가에 대한 알고리즘이라고 생각하면 되는 것이다.
