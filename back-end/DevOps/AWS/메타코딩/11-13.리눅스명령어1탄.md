- 리눅스 명령어를 배운다는 것은 shell을 배우는 것이다. 
- shell을 배운다는 것은 사용자와 컴퓨터와 대화를 나누기 위해서 필요한 언어이다. 
- ls -l 
	- 리스트 자세히보기 
	- d라고 붙어있으면 폴더이다. 
	- - 라고 붙어있으면 파일이다. 
- ls -all
	- 숨김 파일들까지 다 볼 수 있다. 
	- 숨김파일의 역할은 os의 중요한 파일이다. 
- ls -al 
	- 모든 숨김파일까지 자세히보겟다는 것. 
- 특정 명령어 끝에 --help를 붙이면 어떻게 사용하면 되는지를 알려준다. 
- cp 
	- 복사하기 
	- ex) cp a.txt b.txt 
	- a.txt를 b.txt로 복사한다.
- mv 
	- 파일이동하기 / 이름변경하기 
	- ex) mv b.txt user 
		- b.txt를 user폴더로 이동한다.
	- ex) mv a.txt user/c.txt
		- a.txt를 user폴더로 이동하는데, c.txt라는 이름으로 이동시켜라. 
	- ex) mv a.txt d.txt 
		- a.txt를 d.txt로 이름을 변경해라. 
- bin 폴더 
	- 실행 파일이 모여있는 폴더이다. 이 폴더 안에 실행 파일이 모여있음. 
- ln -s 
	- 소프트 링크. 바로가기 파일을 생성하는 것임.
