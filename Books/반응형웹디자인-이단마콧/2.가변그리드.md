
- 그리드 기반의 레이아웃 중 첫 번째 레이어는 다음과 같이 보인다. 

```css
.page {
	width : 960px;
	margin : 0 auto;
}
```

- css에 고정된 너비 값을 설정하고 페이지 가운데에 오도록 한다. 
- 그러나 포터샵에서 제작한 디자인을 좀더 유연하고 비례하도록 변환할 필요가 있다.

```css
.page{
	margin : 36px auto;
	width : 90%;
```
