## 동적프로그래밍에서의 시간복잡도 계산 

```javascript
const foo = (n) => {
	if(n <= 1) return 
	foo(n-1)
}
```
이렇게 foo라는 함수로 호출되면 시간복잡도는 O(n)으로 나오게 된다. 

그런데, 만약 n-2로 바꾸게 되면, 
```javascript
const foo = (n) => {
	if(n <= 1) return 
	foo(n-2)
}
```
시간 복잡도는 O(n/2)가 된다.
