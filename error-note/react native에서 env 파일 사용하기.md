- 먼저 루트 폴더에 ios와 android가 있어야한다. 
	- 기존의 다른 많은 폴더들을 보니, 이 각각의 폴더들이 존재한 것을 볼 수 있었는데, 왜 존재했는지 알 것 같다. 각 디바이스별로 설정해줘야하는 내용들이 있는 것 같고, 관련 설명 파일들을 여기어 넣어두는 것이 아닐까.
